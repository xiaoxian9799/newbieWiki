# Data Structures

## Array

### Declare A New Array and Assign Values to Elements of An Array

```
string[] orderIDs = new string[3];

orderIDs[0] = "A123";
orderIDs[1] = "B456";
orderIDs[2] = "C789";
```

```
string[] orderIDs = {"A123", "B456", "C789"};
```

### Use the Length Property of An Array

```
orderIDs.Length
```
